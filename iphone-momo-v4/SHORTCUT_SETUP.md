
# Apple Shortcuts Setup
1. Open Shortcuts on iPhone.
2. Create shortcut named Import MoMo.
3. Add action: Ask for Input or Receive Shared Text.
4. Add action: Get Contents of URL.
5. URL: https://YOURDOMAIN.vercel.app/api/import
6. Method: POST
7. Request Body JSON: {"text":"Provided Input"}
8. Add to Share Sheet.

Now when you receive an SMS:
- Copy text or Share text
- Run Import MoMo shortcut
- Dashboard updates.
